# 🧘 Mood Journal AI

A simple mood tracking app built using **Python**, **Streamlit**, and **TextBlob**. This tool allows users to write journal entries and receive instant mood analysis using sentiment analysis.

---

## ✨ Features

- 📝 Write about your feelings in a journal-style text box.
- 🤖 Uses AI sentiment analysis (TextBlob) to detect the mood of your writing.
- 🌈 Gives feedback as Positive 😄, Neutral 😐, or Negative 😔.
- 📈 Helps users build emotional awareness and reflection.
